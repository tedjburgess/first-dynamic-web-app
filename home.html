My Social Media Page

I have created this site for my frontend class in my bachelores program in software development. The purpose of it is to practice building a dynamic web application that functions somewhate like a Social media platform.

You will see navigation sections at the top of each page. These include home, posts, and contact. In the posts section you will see post dynamically fetched from the API. Clicking on a username in the posts section will bring you to a model displaying the users profile. The contact page includes a contact form with name and email where inputs will be validated.
